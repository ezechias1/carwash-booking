# CarWash Booking

A car wash service booking website with service listings, testimonials, and an online booking widget.

**Live Demo:** [e-s-services.vercel.app](https://e-s-services.vercel.app)

## Features

- Service listings (Sedan, SUV, Van, Valet)
- Online booking widget
- Testimonials section
- About and Contact pages
- Smooth animations

## Tech Stack

- **Framework:** React 19, Vite
- **Language:** JavaScript
- **Animations:** Framer Motion
- **Icons:** Lucide React

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).
